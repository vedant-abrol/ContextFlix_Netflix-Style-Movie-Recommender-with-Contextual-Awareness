ContextFlix - Personalized Movie Recommender

Welcome to ContextFlix, a context-aware movie recommendation system. This document provides instructions on how to run and use the application.

Project Structure

The project is organized into the following main components:

  • Frontend: React-based web interface for user interaction
  • API: Flask-based backend service that provides recommendations
  • Models: Recommendation algorithms and data processing pipelines

Running the Application

To run the application, you need to start both the API server and the frontend server:

1. Start the API Server

cd movie_recommender
source contextflix_env/bin/activate
cd api
python app.py

This will start the API server on http://localhost:5050.

2. Start the Frontend Server

cd movie_recommender/frontend
source ../contextflix_env/bin/activate
python serve.py

This will start the frontend server on http://localhost:3000.

Note: Both servers need to be running simultaneously for the application to work properly.

Troubleshooting

If you encounter issues, here are some common problems and solutions:

Port Already in Use

If you see an error like "Address already in use", you need to kill the process using that port:

lsof -i :5050 | grep LISTEN  # Check processes using port 5050
kill -9 [PID]             # Kill the process

API Connection Issues

If the frontend can't connect to the API, check the following:

  • Make sure the API server is running at http://localhost:5050
  • Check that CORS is properly configured (the API should return appropriate headers)
  • Use the API diagnostics page to test specific endpoints

Using the Application

Once both servers are running:

  1. Open http://localhost:3000 in your browser
  2. Select a user ID to see personalized recommendations
  3. Explore recommendations based on different contexts (time of day, device type)
  4. View movie details and similar movies

Important: The application requires both the frontend and API servers to be running concurrently.
